# Purah

## Installation

- `python3 -m pip install -U pipenv --pre`
- On Windows, do this instead: `py -3 -m pip install pipenv --pre`
- `git clone` this repository
- `cd purah`
- `pipenv install`

## Running Purah

- `pipenv run hero --test`
